# Event & Ticket Booking System

A Java object-oriented programming project that simulates an event and ticket booking system with customer and admin functionalities.

## Features
- Customer registration and login
- Event browsing and searching
- Ticket booking and cancellation
- Personalized event recommendations
- Admin event management
- Booking statistics and reports

## OOP Concepts Used
- Inheritance
- Polymorphism
- Encapsulation
- Composition
- Association

## Technologies Used
- Java
- Object-Oriented Programming (OOP)
- UML Design

## Files Included
- Java source code
- Project report
- UML and system documentation
